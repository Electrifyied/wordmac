# wordmac
wordmac is a multi-WORD shuffler that utilizes simplicity and fastness in operation
---------------------------------------------------------------------------------------------
for use:
u start by selecting normal or omni
normal offers shuffling in one way
omni offers shuffling in every single way
---------------------------------------------------------------------------------------------
then u choose the file name youd like to create (without extension since its set to .lst)
---------------------------------------------------------------------------------------------
then u put the words/numbers youd like to shuffle
---------------------------------------------------------------------------------------------
then it prompts you with the information of the file and you confirm or deny
